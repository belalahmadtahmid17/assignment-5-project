Important FAQ

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects.

How do you select an element from DOM ?

You can select an element of DOM, by ID (.getElementById), by class name (.getElementsByClassName), by tag name (.getElementsByTagName), by query selector (.querySelector), by all query selector (.querySelectorAll).

What is event delegation in the context of DOM, and why is it useful ?

Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of adding listeners to multiple child elements. The parent then handles events triggered by its children using event bubbling.

How do you manipulate an element's attributes and styles using DOM ?

To manipulate an element's attribute you can use .getAttribute(), .setAttribute(), .removeAttribute or direct access to property such as .src or .alt . To manipulate an element's styles you can use .style and add or remove classes.
